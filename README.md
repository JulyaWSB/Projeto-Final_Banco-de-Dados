# Sistema de Gestão para Clínica Odontológica

O Sistema de Gestão para Clínica Odontológica foi desenvolvido para modernizar o atendimento e a gestão de pacientes de uma clínica odontológica. O sistema permite o agendamento de consultas, o registro de atendimentos, o controle de dentistas e pacientes, além de facilitar o gerenciamento dos horários dos dentistas e a atualização das consultas. Embora não inclua funcionalidades financeiras, a ferramenta visa otimizar o fluxo de trabalho e melhorar a organização do atendimento.
